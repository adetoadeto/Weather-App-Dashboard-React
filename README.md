# Weather App Dashboard

A Weather App Dashboard that provides weather related data of a desired location .

## How to Run the App

- Visit https://todo-app-pgyr.onrender.com

## Technologies Used

- Frontend
    - HTML
    - CSS
    - JavaScript
    - Fetch API 

- Deployment
    - Render

## Key Features Implemented

* Search bar for inputting desired location
* Clean and simple user interface
* Optimal display for mobile and desktop users (responsive)
